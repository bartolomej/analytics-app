### Server-side Application Overview

Java based server application with external restfull API endpoints.
```
 +-- api        -> model oriented api endpoints
 +-- config     -> app configuration files
 +-- db         -> persistance layer
 |   +-- sql    -> sql queries, model schemas,..
 +-- models     -> applicational data structures / models
 +-- scripts    -> sql / shell initialization scripts
 +-- services   -> high level controller components
 +-- utils      -> utility libraries
 ```
